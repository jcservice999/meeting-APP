# 會議軟體 - 使用說明...

## ✨ 主要功能

### 🎤 智慧語音識別
- **混合 AI 模式**：結合 Web Speech API（即時）+ Whisper AI（高準確度）
- **即時轉錄**：說話內容立即顯示
- **AI 修正**：Whisper AI 背景處理，提供更準確的結果
- **多語言支援**：中文、英文自動識別

### 👥 多人協作
- **即時同步**：所有參與者看到相同的字幕
- **說話者標註**：自動標記發言人
- **視覺提示**：說話時顯示綠色框框
- **參與者網格**：清楚顯示所有與會者

### 🔐 權限管理
- **Google 登入**：安全的 OAuth 認證
- **管理員系統**：第一位使用者自動成為管理員
- **審核機制**：管理員審核新成員加入
- **角色分配**：靈活的權限管理

### 💾 完整記錄
- **對話歷史**：完整保存會議內容
- **時間戳記**：每句話都有精確時間
- **可展開檢視**：隨時查看完整對話記錄
- **照片上傳**：自訂個人頭像

- **✅ 完全免費**：使用 Supabase，不需要信用卡！

## 📋 使用流程

### 1. 首次使用（管理員）

1. 訪問網站
2. 點擊「使用 Google 帳號登入」
3. 選擇您的 Google 帳號
4. **自動成為管理員**
5. 進入會議室

### 2. 其他使用者加入

1. 訪問網站並登入
2. 等待管理員批准
3. 批准後進入會議室

### 3. 會議室功能

#### 上傳照片
1. 點擊頂部的 📷 按鈕
2. 選擇照片
3. 點擊「上傳」

#### 切換語言
- 使用頂部的語言選擇器
- 支援中文和英文

#### 查看字幕
- 底部字幕條顯示最新 3 條對話
- 點擊「展開 ▲」查看完整對話記錄
- 每句話都會標註發言人

#### 管理員功能
1. 點擊頂部的 ⚙️ 按鈕
2. 查看待批准使用者
3. 批准或拒絕加入請求
4. 設定其他使用者為管理員

## 🎤 使用提示

### 獲得最佳轉錄效果

1. **環境**：選擇安靜的環境
2. **麥克風**：使用品質良好的麥克風
3. **發音**：清晰發音，避免過快
4. **距離**：保持適當的麥克風距離

### 說話者檢測

- 當您說話時，您的照片會出現**綠色邊框**
- 其他人也能看到誰在說話
- 檢測基於音量，請確保麥克風正常運作

## ❓ 常見問題

**Q: 為什麼我無法加入會議？**  
A: 您需要等待管理員批准您的加入請求。

**Q: 如何成為管理員？**  
A: 第一位使用者自動成為管理員，或由現有管理員設定。

**Q: 轉錄不準確怎麼辦？**  
A: 請確保環境安靜、麥克風品質良好，並清晰發音。

**Q: 可以同時有多少人？**  
A: 建議 10-20 人以內，以獲得最佳體驗。

**Q: 對話記錄會保存嗎？**  
A: 會保存在 Supabase 資料庫中，可隨時查看。

**Q: 需要付費嗎？**  
A: 不需要！完全免費，無需信用卡。
